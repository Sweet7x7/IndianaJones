# Архив Индианы Джонса 🎓📜

**Полное досье экспедиций 1935–1957 годов**

Этот архив содержит секретные документы, личные записи и каталоги артефактов профессора Генри «Индианы» Джонса младшего.

## 🔍 О проекте

Сайт выполнен в стиле полевого дневника археолога. Здесь собраны:
- **Личный дневник** — записи из экспедиций в Перу, Египет, Венецию и Амазонию.
- **Фильмы** — художественные интерпретации приключений.
- **Снаряжение** — проверенная экипировка: куртка, кнут, револьвер, шляпа, сумка и дневник.
- **Каталог артефактов** — от золотого идола Ховитто до хрустального черепа.
- **Полевая телеграмма** — секретный канал связи с Индианой.

## 🛠️ Технологии

- HTML5
- CSS3 (кастомные стили, анимации, адаптив)
- JavaScript (ES6+)
- Font Awesome 6 (иконки)
- Google Fonts (винтажные шрифты)
- Formspree (обработка формы)

## 🖼️ Добавление изображений

Картинки подставляются в трёх местах:

1. **Снаряжение** → `js/equipment.js` (поле `image`)
2. **Фильмы** → `js/films.js` (поле `image`)
3. **Артефакты** → `js/artifacts.js` (поле `image`)

Пример:
```javascript
